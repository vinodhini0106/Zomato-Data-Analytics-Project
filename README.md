# Zomato-Data-Analytics-Project
I recently worked on an end-to-end data analytics project where I analyzed Zomato’s restaurant and order data using SQL and Power BI.

🔍 What I did:

✔Cleaned and transformed raw data using SQL
✔Built KPIs like Total Revenue & Average Order Value
✔Created an interactive Power BI dashboard

📊 Key Insights:

✔Mumbai leads in both revenue and order volume
✔Pricing does not always determine demand
✔Mid-range restaurants often outperform premium ones
✔Customer ratings and sales are not always directly linked

🛠 Tools Used:
SQL | Power BI | Data Analysis

This project helped me strengthen my skills in:
✔ Data cleaning
✔ SQL joins & aggregations
✔ Dashboard design
✔ Business storytelling
