Task 2(3) Find the top 5 cities with the highest number of orders.

city name, highest orders, top 5:

select*from restaurants;
select *from orders;

select r.city, 
count(o.order_id) as no_orders
from orders o
join restaurants r
on r.restaurant_id = o.restaurant_id
group by r.city
order by no_orders desc
LIMIT 5;

Data Aggregation
TASK 3:  Find the average order amount for each city.

AVG ORDER amount, EACH CITY

select* from orders;

select r.city,
avg(o.total_cost) as avg_orders
from orders o
join restaurants r
on r.restaurant_id = o.restaurant_id
group by r.city
order by avg_orders desc;
